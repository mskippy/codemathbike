Unit 3: Game Development

In this tutorial you are going to create a top down shooter game using a game development software called GDevelop. It is an open-source no-code gaming software, but it will introduce you to some basic gaming logic that is needed when you are learning about programming.

Getting started

First we need to create a folder to keep all our files for this project.

  • In your Home Folder create a folder called 3. Game Development
  • Download the Space Shooter Assets file
    • Right click and select download
    • When it has downloaded, click on Open
    • Another window will open, save the Space Shooter Assets folder into your Game Development folder on your Home Drive

Click a lesson to get started:

  • Lesson 2: Player Movement
