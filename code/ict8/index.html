Updates & Announcements

Welcome to ICT 8! We’ll build practical digital skills: file & folder organization, OneDrive workflow, typing, and intro projects in web and game development. Check Teams for assignments and due dates.

  • Sept 2: First class — set up course folders and join our Teams space.
  • Sept 5: Typing baseline check (2 minutes).

What we’ll be doing

  • Getting set up for success: Teams, where to find & submit work.
  • Core digital skills: creating folders, naming files, moving/renaming, downloading & unzipping, uploading to OneDrive.
  • Typing: brief practice each class to build speed & accuracy.
  • Game Development: build a simple platformer in GDevelop.
  • Web Development: clean, accessible pages with HTML & CSS.
  • Digital citizenship: passwords, privacy, organization, positive online presence.
