What we’ll be doing

  • U1 Essential Skills: course expectations, 20-Questions survey, Canva “About Me,” Acceptable Use Agreement, AI & digital-literacy norms.
  • U2 Game Development (GDevelop): setup, movement, collisions, collectibles, enemies, levels, polish, export.
  • U3 Game Promotion (Poster): simple brand kit, layout in Adobe/Canva, copywriting, export for print/web.
  • U4 Web Development (Trailer + Site): plan & edit a 30–60s trailer, build a one-page game site, publish.

We’ll sprinkle in short mini-lessons on smart/ethical AI use, file management, and attribution.
